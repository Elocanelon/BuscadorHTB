Script para buscar diferentes niveles de maquinas en "Hack the box" desarrollado para curso de introduccion de linux.

Archivo de referencia creado por S4vitar!


Usa diferentes parametros tales como:
	u) Descargar o actualizar archivos necesario
	m) Buscar por un nombre de maquina
	i) Buscar por direccion ip
	d) Buscar maquina por dificultad
	o) Buscar maquina por sistema operativo
	s) Buscar maquina por skill
	y) Obtener link de resoucion de la maquina por youtube
	h) Mostrar este panel de ayuda
